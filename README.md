# Aliza Akram – Personal Portfolio

A modern, responsive personal portfolio website built with React, Tailwind CSS, and JavaScript to showcase my skills, projects, education, and professional journey as a Computer Science graduate.

The portfolio highlights my experience in full-stack development, AI-powered applications, database design, and UI/UX design. It features a clean and professional user interface, responsive layouts for all devices, smooth animations, project showcases, technical skills, educational background, and contact information.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Built with React and Tailwind CSS
* Smooth scroll navigation and animations
* Project showcase with technology stacks
* Skills categorized by domain
* Education and professional profile sections
* Contact and social media integration
* Clean, modern, and recruiter-friendly design

## Technologies Used

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* HTML5
* CSS3
* Babel
* Google Fonts
